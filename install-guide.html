装机步骤 ♡

内存 + 硬盘 — 别怕，比看起来简单

准备工作
  1. 关机
  2. 拔电源线
  3. 按 5 秒开机键（放掉残留电）
  4. 摸一下机箱金属外壳放静电
  5. 打开机箱侧板
工具：小号十字螺丝刀一把
0. 先除尘（趁机箱已经打开）

地点：搬到阳台做，灰会飞得到处都是。

喷哪里（按优先级）：

  1. CPU 散热器（缝隙里灰最多）
  2. 机箱所有风扇（前 / 后 / 上）
  3. 显卡风扇（如果有独显）
  4. 底部和前面的防尘网
  5. 空着的 M.2 槽 + DIMM3 内存槽（装新硬件之前先吹）
⚠ 重要：
  • 吹风扇时用手指卡住扇叶，别让它转（反转会发电烧主板）
  • 罐子竖着拿，斜了会喷液体冻坏元件
  • 短促按 1-2 秒，距离 15-20 厘米
  • 别拆电源（PSU），只从外面吹通风口
  • 吹完等 5 分钟再装机，让冷凝水汽挥发
1. 装内存（Colorfire 16GB）

位置：DIMM3 槽

4 个内存槽从 CPU 那一侧数过去——你现有的 Colorful 在 DIMM1，跳过 DIMM2，新的插 DIMM3。

  1. 把 DIMM3 两边的白色卡扣往外扳开
  2. 内存条上有个缺口，对准槽里的凸起
    （只有一个方向能插，反了插不进，不要硬来）
  3. 两个大拇指均匀往下按
  4. 听到"咔哒" + 卡扣自动弹回扣住 = 完成 ✓
2. 装硬盘（Mars 980 Blade）

主板有 2 个 M.2 槽：

  • 上面 M2A_CPU = PCIe 4.0（快槽）
  • 下面 M2B_SB = PCIe 3.0（慢槽）

目标：Mars 980 装快槽，金士顿挪到慢槽

  1. 找到上面那个 M.2 槽（靠近 CPU 散热器）
  2. 拆掉金士顿尾部的小螺丝
    别丢这颗螺丝！
  3. 金士顿会自动翘起来，拿出来
  4. 把金士顿装到下面那个 M.2 槽：
    • 斜着 30° 插进去，金手指插到底
    • 按下平贴主板
    • 上小螺丝固定
  5. 把 Mars 980 装到上面那个 M.2 槽：
    • 斜着 30° 插进去，金手指插到底
    • 按下平贴主板
    • 用刚才那颗小螺丝固定
3. 开机检查

盖机箱 → 插电源 → 按开机键

情况 1：直接进了 Windows ✓

什么都不用做。进系统后检查：

  • Win + Pause 看内存是不是 32GB
  • 此电脑右键 → 管理 → 磁盘管理
    找到 Mars 980（显示"未分配"）
    右键 → 新建简单卷 → 分配盘符（E 盘）
情况 2：黑屏没进系统

别慌！只是 BIOS 启动顺序乱了：

  1. 重启时狂按 Delete 键进 BIOS
  2. 找 Boot 启动项页面
  3. 第一启动改成 KINGSTON SNV2S1000G
  4. F10 保存退出 → 搞定
常见担心 FAQ
Q: 内存/SSD 反方向插了怎么办？
反了根本插不进去，按不下去。感觉不对就翻面再试。只有硬掰才会坏，别硬来。
Q: 螺丝拧不动？
别用力。螺丝刀对准凹槽，垂直按下再转。滑了就停，换角度再来。
Q: 静电会烧坏吗？
拔电源 + 摸机箱外壳放电就够了。别穿毛衣装机就行，不用买静电手环。
Q: BIOS 看不懂英文？
Gigabyte BIOS 有中文，进去后角落能切换语言。
加油！其实就是插和拧，不难 💪
